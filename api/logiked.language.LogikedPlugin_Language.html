    Class LogikedPlugin_Language

    Scriptable object unique à chaque plugin permetant la configuration des assembly Logiked. Accesible via le code par un accesseur statique Instance afin d'être disponible dans tout le projet. Accessible dans l'inspecteur unity dans le dossier Resources/Logiked

    Les paramètres du patch de langues par défauts sont configurables ici (Langue, Auteur, Version...)
